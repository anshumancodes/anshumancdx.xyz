# anshumancdx.xyz

- **Library** [reactjs](https://react.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com)
- **Database & auth**: [Firebase](https://firebase.google.com/)


## Running Locally

`
git clone https://github.com/anshumancodes/anshumancdx.xyz.git
cd anshumancdx.xyz
npm install
npm run dev
 

`
